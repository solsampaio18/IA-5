Neural Network

Compilação:
 - Na pasta onde se encontra o ficheiro, efetuar o seguinte comando: javac Test.javac
 - Para executar o programa, efetuar o seguinte comando: java Test.javac

Utilização:
 - Para alterar a learning rate, basta entrar no ficheiro Test.java onde tem learning rate alterar, depois tem de recompilar.
 - Para alterar o número de hidden nodes basta mudar o segundo valor onde tem o construtor da NeuralNetwork chamada brain, depois recompilar.
